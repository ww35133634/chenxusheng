ls: 展示出当前路径(目录)下所有的文件/文件夹

ctrl shift +: 放大终端字体

ctrl -: 缩小终端字体

pwd: 显示当前所处位置的路径信息

根目录: 最顶级目录

cd: 进入一个目录

cd ..: 进入上级目录

cd -:进入上一次在的目录

cd ~:进入家目录

clear: 清洗终端 ctrl + L

tree: 查看当年路径之下的目录树

ctrl+c: 停止服务端

touch: 创建一个文件

rm: 删除文件

mkdir: 创建文件夹

rmdir: 删除文件夹

rm -r : 删除文件夹

^(上键):翻到历史命令

下键: 翻到历史命令

斜杠:/

反斜杠:\

cp: 复制一个文件 到 另外一个地方

cp -r:复制一个文件夹 到另外一个地方

mv: 更换位置, 重命名

history: 查看历史命令

ls --help:查看文档

man ls:查看文档

ls -h:当前路径下详细信息展示
1:(文件/文件夹)权限
2:硬链接数量
3,4:用户名,分组
5:容量
6,7,8:最后一次修改它的时间
9:名称

ls -h: 显示具体容量

ls -a: 显示隐藏文件/文件夹

touch .文件名/文件夹名:创建隐藏文件/隐藏文件夹

ls -alh: 总汇

sudo 命令语句:用管理员的身份去执行改命令

sudo -s: 变成管理员

exit: 退出管理员身份
